## 서비스 기획 포트폴리오

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **서비스명** | Dandori (단도리) |

## 2. 기획 배경

### 2-1. 문제 인식

마케팅 업무 현장에서는 다음과 같은 비효율이 반복되고 있었다.

- **도구 파편화**: 프로젝트 일정은 Notion, 예산은 Excel, 설문은 Google Forms, SNS 성과는 각 플랫폼에 흩어져 있어 통합 보고가 어렵다.
- **수작업 지표 취합**: 인스타그램·유튜브·틱톡 등 플랫폼별 성과를 매번 수동으로 복사·붙여넣기 하여 보고서를 작성한다.
- **검색 트렌드 파악의 어려움**: 키워드 시장조사를 위해 네이버 DataLab, 광고센터, 포털 검색 등을 별도로 열어 비교해야 한다.
- **팀 협업 단절**: 팀원별 역할·진행 상황이 한눈에 보이지 않아 프로젝트 오너가 상시 수동 업데이트해야 한다.
- **고객 조사 기능 부재**: 별도의 설문 솔루션을 쓰거나 외부 링크로 이동시켜야 하여 데이터 일관성이 깨진다.

### 2-2. 기획 계기

B2C/B2B 마케팅 캠페인을 반복 운영하며 "기획부터 결과 보고까지 한 곳에서 처리할 수 있는 도구"에 대한 수요를 직접 체감했다. 기존 솔루션(Asana, Notion, Monday.com 등)은 범용 프로젝트 관리 도구로, 마케팅 도메인 특화 기능(SNS 지표 자동 연동, 키워드 분석, 설문 빌더 내장)이 없었다.

## 3. 시장 조사

### 3-1. 경쟁 솔루션 분석

| 솔루션 | 강점 | 마케팅 특화 한계 |
|--------|------|-----------------|
| Notion | 문서·DB 유연성 | SNS 지표 연동 없음, 설문 빌더 없음 |
| Asana | 태스크 관리, 타임라인 | 마케팅 보고서 특화 기능 없음 |
| Monday.com | 시각화 대시보드 | 비용 높음, 국내 검색 API 미지원 |
| Google Forms | 무료 설문 | 프로젝트 연동 없음, 브랜딩 제한 |
| Sprout Social | SNS 분석 | 고가, 국내 플랫폼(네이버) 미지원 |
| 블랙키위 / 네이버 광고센터 | 키워드 트렌드 | 독립 서비스, 프로젝트 연계 불가 |

### 3-2. 기회 발견

- 국내 마케팅 팀 특화: **네이버 DataLab + 검색광고 API** 연동을 지원하는 프로젝트 관리 도구가 없다.
- **풀스택 워크플로우** 지원 도구의 공백: 기획 → 실행(설문/콘텐츠) → 성과 분석을 단일 플랫폼에서 제공하는 국내 SaaS 부재.
- **중소형 마케팅 팀** 세그먼트: 엔터프라이즈 솔루션을 도입하기 어려운 팀을 위한 합리적 가격대의 대안 필요.

### 3-3. 타겟 사용자

| 페르소나 | 역할 | 주요 Pain Point |
|----------|------|----------------|
| 마케팅 매니저 | 캠페인 전체 관리 | 산출물·예산·KPI를 개별 도구에서 취합하는 반복 작업 |
| 콘텐츠 크리에이터 | SNS 콘텐츠 제작 | 게시물 성과를 수동으로 정리·보고 |
| 리서치 기획자 | 시장·고객 조사 | 설문 제작-배포-분석 도구 파편화 |
| 에이전시 PM | 다수 브랜드 캠페인 운영 | 클라이언트별 보고서 생성 공수 과다 |

## 4. 목표 수립

### 4-1. 비즈니스 목표

1. 마케팅 프로젝트 운영 도구를 **단일 플랫폼**으로 통합하여 도구 전환 비용 제거
2. 데이터 자동 수집(SNS 지표, 키워드 트렌드)으로 **보고서 작성 시간 70% 단축**
3. 국내 마케팅 팀이 즉시 사용 가능한 **한국어 특화 플랫폼** 제공

### 4-2. 제품 목표 (MVP 기준)

| 목표 영역 | 세부 목표 |
|-----------|-----------|
| **프로젝트 관리** | 목표/예산/일정/이슈를 하나의 프로젝트 워크스페이스에서 관리 |
| **설문·폼 빌더** | 코딩 없이 커스텀 설문 제작·배포·응답 수집 가능 |
| **SNS 성과 추적** | URL 하나로 산출물 등록 → 지표 자동 수집·집계 |
| **키워드 인텔리전스** | 네이버 DataLab + 검색광고 API로 실시간 검색량 분석 |
| **통합 인사이트** | 프로젝트 전체 KPI 달성률을 대시보드 한 화면에서 확인 |

### 4-3. 성과 지표 (KPI)

- 신규 프로젝트 생성 수 / 월
- 설문 응답 수집 건수
- SNS 산출물 자동 연동 비율
- 보고서 CSV 내보내기 사용 빈도
- 사용자 재방문율

## 5. 정책 수립

### 5-1. 접근 권한 정책

| 역할 | 권한 |
|------|------|
| **administrator** | 전체 사용자 관리, 사이트 설정, 공지·릴리즈 노트 관리 |
| **editor** | 프로젝트·폼 생성·수정, 팀원 초대 |
| **project member (owner/manager/member/viewer)** | 프로젝트 단위 역할 기반 접근 제어 |
| **anonymous** | 공개 설문 URL 접근·제출만 허용 |

- 모든 데이터는 Supabase RLS(Row Level Security)로 소유자 단위 격리
- 인증 세션: Supabase Auth 쿠키 기반, 미들웨어로 `/dashboard`, `/projects` 보호

### 5-2. 데이터 보관 정책

- 응답 데이터: 프로젝트 오너 삭제 전까지 보관
- 파일 업로드: Supabase Storage public 버킷, CDN 서빙
- API 연동 캐시: SNS 지표 1시간 주기 자동 갱신 + 관리자 수동 새로고침 제공

### 5-3. 개인정보 처리 방침

- 설문 응답자 이메일은 알림 발송 후 별도 마케팅 목적 활용 금지
- 개인정보처리방침·이용약관 페이지 제공 (`/privacy`, `/terms`)
- 이메일 발송: Resend API 사용, 수신 동의 기반

### 5-4. 서비스 안정성 정책

- 설문 마감일(deadline) 및 최대 응답 수(max_submissions) 초과 시 자동 접수 차단
- 외부 API 연동 실패 시 마지막 성공 데이터 유지 (graceful degradation)
- 웹훅 발송 실패는 로그 기록 후 재시도하지 않음 (단방향 알림)

### 6-2. 핵심 기능 상세

#### [1] 폼 빌더 (Form Builder)

- **17종 필드 타입**: 텍스트, 이메일, 긴 텍스트, 체크박스, 셀렉트, 라디오, 별점, 섹션(다중 페이지), HTML, 이미지, 지도, 유튜브, 표 등
- **드래그앤드롭 인터페이스**: @dnd-kit 기반 필드 순서 변경
- **조건 분기 로직**: 라디오 응답에 따라 다음 섹션 분기
- **커스텀 테마**: 8가지 색상 프리셋 + 커스텀 컬러 적용
- **다국어 지원**: 한국어·영어·일어·중국어
- **SEO 설정**: 제목·설명·OG 이미지 커스터마이징
- **이메일 알림**: 관리자 + 응답자 확인 메일 자동 발송
- **응답 통계**: 필드별 파이차트·막대차트·평점 분포 시각화
- **CSV 내보내기**: Excel 호환 UTF-8 BOM 인코딩

#### [2] 프로젝트 워크스페이스

- **목표·KPI 플래너**: 정량/정성 지표별 목표값·실적값·달성률·가중치 관리
- **예산 플래너**: 항목별 예산 배분, 최소·최대 금액, 가중치 설정
- **간트 차트**: 마일스톤 시각화, 진행률(0~100%) 추적
- **칸반 태스크**: 상태별 태스크 보드
- **이슈 트래커**: 버그·제안·문의 유형, 긴급도·상태 관리
- **팀 멤버 관리**: 역할(owner/manager/member/viewer) 기반 초대

#### [3] 산출물 & 인게이지먼트 (Outputs)

- **Deliverables (SNS 지표 자동 수집)**
  - 지원 플랫폼: Instagram, YouTube, TikTok, Twitter, Facebook, Blog
  - URL 입력 시 플랫폼 자동 감지·메타데이터 파싱
  - 조회수·좋아요·댓글·공유 수 자동 수집
  - **1시간 주기 자동 갱신** + 수동 새로고침 버튼
- **Clippings (보도자료 아카이브)**
  - og:title·og:description·og:image·발행일 자동 파싱
  - 출처·키워드 검색 필터

#### [4] 인사이트 대시보드 (Insights)

- 전체 조회·노출 합산 KPI 카드
- 전체 인게이지먼트(좋아요+댓글+공유) 집계
- 보도자료 노출 건수
- 목표 대비 달성률 시각화
- 주요 클리핑 목록 요약

#### [5] 블루베리 (Blueberry) — 키워드 인텔리전스

- **네이버 DataLab API**: 검색 트렌드 지수(시계열 그래프)
- **네이버 검색광고 API**: 실제 월간 검색량
- 최대 5개 키워드 동시 비교
- 기간 필터: 1·3·6·12개월
- 연관 키워드 테이블
- CSV 내보내기
- 구글 탭: 비교 분석용 참고 데이터 제공

### 6-3. 기술 아키텍처

```
Client (Browser)
  └── Next.js 16 App Router (SSR + CSR 혼합)
        ├── Server Components → createServerClient() [Supabase]
        ├── Client Components → createClient() [Supabase]
        └── Public Forms    → createPublicClient() [Supabase Anon]

API Layer
  └── Next.js Route Handlers
        ├── /api/forms/*        폼 CRUD
        ├── /api/submissions/*  응답 수집·조회
        ├── /api/projects/*     프로젝트 관리
        ├── /api/deliverables/* SNS 지표 수집
        ├── /api/clippings/*    OG 메타 파싱
        ├── /api/blueberry/*    네이버 API 프록시
        └── /api/admin/*        관리자 기능

Data Layer
  └── Supabase (PostgreSQL + RLS)
        ├── 23개 마이그레이션 관리
        ├── Storage (이미지·파일 업로드)
        └── Auth (세션 관리)

External Services
  ├── Resend API        이메일 발송
  ├── Naver DataLab API 검색 트렌드
  └── Naver 검색광고 API 검색량 데이터
```

## 7. 개발 일정

### Phase 1 — 핵심 인프라 & 폼 빌더 (완료)

| 기간 | 작업 |
|------|------|
| 환경 구성 | Next.js 16 + Supabase 셋업, 인증 미들웨어, DB 스키마 초기 설계 |
| 폼 빌더 MVP | 드래그앤드롭 빌더, 17종 필드, 조건 분기, 테마 |
| 응답 수집 | 공개 폼 URL, 응답 저장, 이메일 알림 |
| 응답 통계 | 필드별 차트, CSV 내보내기 |

### Phase 2 — 프로젝트 워크스페이스 (완료)

| 기간 | 작업 |
|------|------|
| 프로젝트 CRUD | 생성 위저드, 개요 페이지, 팀원 관리 |
| KPI 플래너 | 목표 설정, 실적 입력, 달성률 계산 |
| 예산 플래너 | 항목별 예산 배분, 가중치 |
| 간트 차트 | 마일스톤 시각화, 진행률 |

## 8. 기대 효과

### 8-1. 사용자 관점

| 효과 | 내용 |
|------|------|
| **업무 효율화** | 5~7개 분산 도구를 1개 플랫폼으로 통합, 도구 전환 비용 제거 |
| **보고 자동화** | SNS 지표 자동 수집으로 주간·월간 보고서 작성 시간 대폭 단축 |
| **의사결정 가속** | 키워드 트렌드·SNS 성과·KPI 달성률을 실시간으로 한 화면에서 확인 |
| **협업 투명성** | 팀원별 역할·진행 상황을 프로젝트 단위로 공유, 커뮤니케이션 비용 감소 |
| **고객 조사 내재화** | 별도 설문 솔루션 없이 프로젝트 내에서 설문 제작·배포·분석 일괄 처리 |

### 8-2. 비즈니스 관점

| 지표 | 기대 효과 |
|------|----------|
| 보고서 작성 시간 | 수작업 대비 약 70% 단축 |
| 도구 구독 비용 | 분산 SaaS 다중 구독 대비 비용 절감 |
| 데이터 정합성 | 플랫폼 통합으로 수작업 오류 0% 지향 |
| 온보딩 속도 | 통합 워크스페이스로 신규 팀원 적응 시간 단축 |

### 8-3. 제품 확장 가능성

- **API 생태계 확장**: TikTok·Pinterest 등 신규 SNS 플랫폼 연동

- **클라이언트 포털**: 에이전시-클라이언트 간 실시간 리포트 공유
- **팀 플랜 과금**: 무료 개인 플랜 + 팀 플랜 구독 모델 도입
- **화이트라벨**: 에이전시용 브랜딩 커스터마이징 옵션
